# MoyaDemo
A demo project for Moya third-part framework in Swift.

# 要求
- Xcode8.3.2+
- iOS8+
- Swift3.2+

# 使用
先用Carthage安装Moya库
> cd 工程目录

> carthage update --platform iOS

如果没有安装Carthage，参照：https://github.com/Carthage/Carthage
